Alexa Skills Kit SDK For PHP

  • PHP

Status: In Development

At present, there isn't a PHP SDK for the Alexa Skills Kit. The aim of this project is to fill this void and help reduce the overhead of creating skills for the Amazon Echo line of devices, enabling developers to concentrate on the features and functionality of their skill.

The SDK is intended to utilise PHP and may also include functionality to enable the storage of session data and keep state when a user ends the skill abruptly allowing the interaction to continue later.

All source code will be freely available on GitHub allowing further collaboration to help improve the SDK.

Dundee Gig Guide Alexa Skill

  • PHP

Status: Live

There are many reports of the demise of live music within Dundee, however, there are events happening on an almost daily basis.

The aim of this skill was to enable anyone to be able to specify a day and find out what live events are happening then.

The skill utilises a PHP script that calls to the ents24.com API.

Moving forward there is scope to improve the skill by introducing features such as allowing the user to specify the genre of an event they are interested in or to follow certain artists and receive notifications if and when an event is scheduled.

Wacken Information Alexa Skill

  • Node.js
  • Lambda
  • Alexa Skills Kit

Status: Live

Having visited the Wacken festival on numerous occasions I thought I would look into creating a skill that would help people keep up to date with the latest information about the festival as well as find out a bit about the past.

At present the skill has the following features:

  • Ability to ascertain if a band has performed in the past
  • Ability to ascertain when a band performed in the past
  • Ability to ascertain who is scheduled to play the next scheduled festival
  • Ability to tell the user how long until the next festival

At present, the skill is hosted on the AWS Lambda service utilising Node.js and the Amazon Skills Kit. The data is stored in JSON files within the lambda function, however, I am currently working on migrating this over to DynamoDB to enable ease of updates.
